# dailysum

タイムスタンプとともに量や自由入力の種別を記録するための小さなWebアプリです。今日の合計の一目での確認、履歴グラフ、クイック入力ショートカット、CSVデータ管理機能を提供します。

ライブデモ: [https://code4fukui.github.io/dailysum/](https://code4fukui.github.io/dailysum/)

## 機能

- **データの記録**: 現在のタイムスタンプとともに量と任意の種別を記録します。
- **ダッシュボード統計**: 今日の合計量、最新の記録、記録日数といった主要な指標を一目で確認できます。
- **データの可視化**: 2つのグラフで傾向を分析できます。種別ごとの日次合計の積み上げ棒グラフと、直近1週間の1日内増加を示す折れ線グラフです。
- **クイック入力**: 直近のユニークな量と種別の組み合わせ5件に対して、自動的にショートカットボタンを作成します。
- **履歴管理**: すべての記録を表形式で表示し、タイムスタンプの編集や個別の記録の削除が可能です。
- **データのポータビリティ**: すべてのデータをCSVファイルとしてインポートおよびエクスポートできます。
- **ローカルファースト**: すべてのデータはブラウザの `localStorage` にプライベートに保存されます。サーバーやアカウントは必要ありません。

## ファイル

- `index.html` - アプリケーションのHTML構造とUIレイアウト。
- `style.css` - 色、レイアウト、グラフの外観を含むすべてのUIスタイリング。
- `main.js` - データ処理、`localStorage` との連携、グラフの描画、CSV操作を含むコアアプリケーションロジック。

## 使い方

最新のWebブラウザで `index.html` を開くか、上記のライブデモのリンクを使用してください。

## データ形式

- **保存**: データは `daily-sum-records` というキーで `localStorage` に保存されます。
- **CSV形式**: CSVファイルは `timestamp,type,amount` の3列を使用します。
